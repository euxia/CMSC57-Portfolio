Basics of Counting technically is a prerequisite in understanding the principles of combinatorics. There are fundamental rules to be remembered for Basics of Counting. Product Rule and Sum Rule. Product rule theory states that; suppose a procedure can be broken down into a sequence of two tasks. There are n1 ways to do the first task and n2 ways to do the second task. Then there are n1âˆ™n2 ways to do the procedure.

Product Rule Example: Suppose I have a store with hats featuring two designs: one with stars and one with stripes. For the star hats, I have 2 styles and each style comes in 2 colors. For the stripe hats, I have 2 styles and each style comes in 3 colors. I want to buy a star hat and a stripe hat.

First, calculate the total number of star hats: 2 (styles) x 2 (colors) = 4 (star hats)

Next, calculate the total number of stripe hats: 2 (styles) x 3 (colors) = 6 (stripe hats)

Next, calculate the total number of stripe hats: 4 (star hats) x 6 (stripe hats) = 24 (total hat options)

Sum rule, on the other hand, states that; If a task can be done either in one of n1 ways or in one of n2 ways to do the second task, where none of the set of n1 ways is the same as any of the n2 ways, then there are n1 + n2 ways to do the task.

Sum Rule Example: If there are 5 ways to get to school by bus and 3 ways by bike, and I can choose only one mode of transportation, then there are 5 + 3 = 8 ways to get to school.

These fundamental concepts must be understood because it is the foundation of many topics to come. By combining the two concepts, sum rule and product rule, we can solve real life problems that already involved combinatorics.

"How many bit strings of length eight either start with a bit or end with the two bits?"

The number of bit strings of length eight that start with a 1 bit is 27 = 128. The number of bit strings of length eight that start with the bits 00 is 26 = 64. The number of bit strings of length eight that start with a 1 bit and end with the bits 00 is 25 = 32. Using the inclusion-exclusion principle, the total number of bit strings of length eight that start with a 1 bit or start with 00 is 128 + 64 - 32 = 160.
